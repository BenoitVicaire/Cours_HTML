Pourquoi apprendre le HTML/CSS?

Le HTML (Hypertext Markup Language) est un language de balisage qui permet de créer des pages web.

Le CSS (cascading style sheet) quant à lui permet de mettre en forme lesdites pages web.

A eux deux nous pouvons réaliser toutes les pages web en circulation sur internet.

Apprendre le HTML/CSS c'est apprendre à écrire sur la toile pour laisser sa trace dans le monde digitale!

Les balises

Le HTML est un language de balisage, ce qui implique l'usage de balises!

Celles ci servent à découper et styliser notre page

Elles sont composé d'une balise ouvrante: <#> et d'une balise fermante marquées par le /: </#>

Les bonnes pratiques!

Il est important de rendre son code maintenable, souvent vous devrez vous replonger dans votre code des mois voir des années après l'avoir écris, parfois vous devrez reprendre le projet d'un autre webdev.

Affin de ne pas trop s'arracher les cheveux il est important de construire son code de manière logique et ordonné ainsi que de respecter certaines convention et commenclature

Pour la partie HTML:

  • Commenter son code affin d'indiquer ce que tel bout de code fait, l'idée est simplement d'avoir une vue grossière de ce qui se passe affin de s'y retrouver.
  • Aerer son code, un code compact sans retour a la ligne ne fait aucune difference pour un ordinateur, mais pour nous pauvres humain, c'est la migraine assurée.
  • Indenter son code affin d'ameliorier la visibilité
  • Respecter la nomenclature en vigueur dans votre équipe, ou une convention de nommage classique

Pour le CSS

  • Les commentaires sont toujours de mises
  • L'indentation aussi
  • Diviser son css en plusieurs fichiers afin de s'y retrouver plus facilement
